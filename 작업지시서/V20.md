# 작업지시서 V20

**작성일:** 2026-04-03
**목적:** 사이드바 UI 단순화 및 다이어그램 정보 레이블 추가

---

## 1. 개요 및 변경 사항 파악

현재 우측 사이드바는 [Parameter]와 [Analysis Report] 두 개의 탭으로 구성되어 "Dots Navigation"을 통해 전환하도록 되어 있습니다. 이를 단순화하여 분석 후에도 페이지 전환 없이 파라미터 조작 화면만 유지되도록 개편하고, 다이어그램에 "VIEWPOINT"라는 명시적인 레이블을 추가합니다.

## 2. 세부 수행 계획 (`App.tsx`)

### ① `SitePlanDiagram` 레이블 추가
- 다이어그램 컨테이너 좌측 상단에 `VIEWPOINT` 텍스트 추가.
- 스타일: `top: 12px`, `left: 12px`, `font-mono`, `text-[9px]`, `font-bold`, `uppercase`, `tracking-widest`, `opacity-40` 적용.

### ② 사이드바 탭 및 페이지 전환 로직 제거
- `activeTab`, `setActiveTab` 상태 변수 제거.
- 상단 "Dots Navigation" (점 두 개 버튼) 섹션 삭제.
- 사이드바 렌더링 시 `activeTab === 'create'` 조건부 렌더링을 삭제하고, 항상 파라미터 조작(SitePlanDiagram + 슬라이더들) 화면만 나타나도록 고정.
- 기존 "Analysis Report" (PHASE 3 Protocol DNA 등) 탭 내용은 화면에서 제거.
